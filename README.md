## Алихан

Backend / Full-Stack разработчик. Проектирую серверную архитектуру, разрабатываю масштабируемые веб-приложения и внедряю DevOps-практики в процесс разработки. Придерживаюсь принципа премиального минимализма — чистый код, понятные интерфейсы, минимум лишнего.

---

### Специализация

- **Backend** — проектирование API на FastAPI и Go, работа с базами данных, отказоустойчивые серверные системы
- **Frontend** — клиентская часть на React, интеграция с REST API
- **Data & ML** — построение и встраивание моделей на scikit-learn в продуктовый backend
- **DevOps** — контейнеризация через Docker, настройка сред разработки, автоматизация развертывания
- **AI-assisted разработка** — использую LLM как часть рабочего процесса: ревью кода, проектирование архитектуры, ускорение рутинных задач при сохранении контроля над финальным решением

---

### Сейчас в разработке

**Incident Predictor** — система прогнозирования инцидентов: модель на scikit-learn под капотом, FastAPI на backend, React на фронте, всё развёрнуто в Docker-контейнерах. Часть архитектурных решений и код разрабатываются в связке с LLM.
